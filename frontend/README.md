
# ProjectTrack Frontend (React + Vite)

## Quick Start
```
npm install
npm run dev
```
- The app expects the backend at `http://localhost:3001`. To override, create `.env` with:
```
VITE_API_BASE=http://localhost:3001
```
- Login as admin using **admin@demo.com / Pass@123** (after seeding backend).
- Create employee users from Admin → Users. Then log out and log in as employee to see Employee dashboard.
